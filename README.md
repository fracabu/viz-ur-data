<a name="english"></a>
## :gb: English

### Overview

Modern web application for creating interactive data visualization dashboards in Marketing, Healthcare, and Finance sectors. Features dark/light mode and responsive design.

### Features

- **Multi-Sector Dashboards** - Marketing, Healthcare, Finance
- **Dark/Light Mode** - Theme switching support
- **Responsive Design** - Works on all devices
- **Interactive Charts** - Dynamic data visualizations
- **Modern Stack** - Next.js + TypeScript

### Sectors

| Sector | Features |
|--------|----------|
| Marketing | Campaign analytics, conversion tracking |
| Healthcare | Patient data, facility metrics |
| Finance | Revenue tracking, financial KPIs |

### Quick Start

```bash
git clone https://github.com/fracabu/viz-ur-data.git
cd viz-ur-data
npm install
npm run dev
```

Access at `http://localhost:3000`
